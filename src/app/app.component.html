Connect Four


Connect Four

The aim of the game is to connect four or more of your pieces vertically, horizontally or diagonally. The pieces always fill the lowest available spot in the column (physical game uses gravity for this). With perfect play, the player to go first can always force a win.

Monte Carlo Algorithm

The Monte Carlo algorithm makes decisions based on random actions and statistics. In this example it has been used to solve for the problem of Connect Four.

Step 1

For every possible move from a position, simulate n games, in this case n=200. Connect Four has up to m=7 possible moves from a given position as there are 7 columns to choose from. Therefore, the maximum number of games we have to simulate is n * m = 7 * 200 = 1400.
If instead we tried to calculate every possible game from a given position, worst case we would have to calculate 27*6 = 242 = 4,398,046,511,104 games.
The simulations are just a series of random choices and are played until either side wins or it's a draw. A draw gives 0.5 points, a win 1 point and a loss 0 points. Remember, we make a legal move first, then simulate the random games.

Step 2

Once we have simulated random games for every legal move, we can pick the move with the highest value. This approach is simple, but works well enough for our use case.

Extra

If more performance is required, an upper confidence bound can be applied to the search tree to guide the search.
